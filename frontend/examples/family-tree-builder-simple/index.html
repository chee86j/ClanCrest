Simple Family Tree Builder

Minimal configuration example

How to Use

This example shows the FamilyTreeBuilder component with minimal configuration. The component provides:

  • Form-based person addition
  • Real-time tree visualization
  • Statistics dashboard
  • Save/load functionality
  • Export capabilities
// Basic usage const builder = new FamilyTreeBuilder(container, { api_base_url: 'http://localhost:3001/api', auto_save: true })
